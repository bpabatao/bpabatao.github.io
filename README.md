# bpabatao.github.io

Personal portfolio of Benedict Pabatao - Lead Platform Engineer.

Next.js 15 (static export) + React 19 + Tailwind CSS v4 + Motion, deployed to GitHub Pages via Actions.

## Develop

```
npm install
npm run dev
```

## Build

```
npm run build   # static export to out/
npm start       # serve out/ locally
```

Content lives in `src/data/` (typed). Deploys automatically on push to `main`.
